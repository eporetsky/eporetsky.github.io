eporetsky.github.io

**Beautiful Jekyll** (remote theme) + Jekyll 4 — see `Gemfile` and `_config.yml`.

Local: install [Pixi](https://pixi.sh), then `pixi install`, `pixi run install`, `pixi run serve` (or `pixi run dev` for install + live reload).

Deploy: GitHub Actions workflow in `.github/workflows/` — set repository **Pages → Build: GitHub Actions**.

Details: `notes.example.txt`.
